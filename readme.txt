Reskillify.com platform plugin for WooCommerce. 

== Description ==

This plugin allow you to connect woocommerce with reskillify.com platform to link courses with woocommerce products.

Contact us through email info@reskillify.com if you have any questions or comments about this plugin.

== Installation ==

Make sure that you already have WooCommerce plugin installed and activated.

**Step 1:**

- Login to your *WordPress Dashboard*
- Go to **Plugins > Add New**
- Search **Reskillify**

**Step 2:**

- **Activate** the plugin through the 'Plugins' screen in WordPress.

**Step 3:**

- Go to Admin backend and click on **Reskillify.com**

**Step 4:**

- Fill in your **API Key** and **Secret Key** that you have generated from reskillify.com platform.
- Click **Save** to save changes.

Contact us through email info@reskillify.com if you have any questions or comments about this plugin.
